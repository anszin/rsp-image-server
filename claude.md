## F&B pos 매뉴 관리 플랫폼

## 기술스택
백앤드 : java, openjdk 17, Springboot, Gradle
프론트 : react

## 기능
1. 각 매장별 POS, KIOSK, QR 결제 등에서 사용할 메뉴 정보(상품코드, 이미지, 상품명, 속성들)를 등록 하는 시스템
2. 각 단말기가 메뉴를 받아 갈 수 있는 이미지 CDN 같은 구조(API 제공), 단말기는 본 프로젝트 범위 아님
3. 향후 QR오더도 구현 할 수 있는 구조 설계
4. 메뉴 등록은 키오스크/pos 화면에 실제 보이는 것처럼 화면 구성
5. 점포id 기반 메뉴 저장/수정/삭제
6. POS/KIOSK 화면 레이어웃 설정(가로형/세로형, 메뉴 배치 n x n) 에따라 실시간 메뉴 화면 보여줌
7. 페이지 넘어가는 것도 고려
8. 카테고리도 설정 가능하게
9. 관리자 페이지 추가 필요 : 고객사 정보(고객사명, 코드) 추가/삭제/수정
10. 관리자 페이지에서 매장 정보 세팅할수 있게(매장명, 메장코드, 매장 소개 문구, 사용구분)
11. 실시간 메뉴 판에 고객사별로 사용 중인 매장 리스트가 보이고, 하위에 메뉴들을 넣을수 있게
12. 각 메뉴판의 자리를 지정해서 편집할수 있게
13. 고객사 먼저 선택하고(나중에는 로그인으로 분리), 해당 고객사의 매장을 선택하도록 
14. 한 매장에서 여러 메뉴판을 운영가능하게 메뉴판 선택/편집 기능 추가
15. 화면 레이아웃 정보도 메장마다 저장되게
16. 화면 레이아웃이 오른쪽에 보이면 좋을듯
17. 한 매장안에는 KIOSK, POS, QR오더가 모두 존재 할수 있고, 이에 맞는 메뉴 배치를 다르게 할수있음
